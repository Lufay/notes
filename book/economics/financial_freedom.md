财富自由


# 为什么需要财富
1. 为了安全感
安全感来自于一个人对未来的预见和准备，使其在困境中保持镇定和自信
2. 为了幸福快乐
钱带来快乐或者幸福感的效应也满足边际效用递减的，所以同样的钱，对于低收入者，直接给与幸福感更高，而对于高收入者，换成优质体验幸福感更高。
当然人对快乐感受的阈值也是会同步增高，但大多数人的资产增长是线性的，所以要么引入大方差的意外收入强化刺激，要么引入多维度的满足感体验进行差异化竞争。
不要将快乐或幸福建立在单一的依赖之上，金钱不行，房子不行，事业不行，爱情不行，友情更不行，所以反脆弱就不要自己断绝更多的幸福支撑，而是要建立更多的羁绊。
3. 为了更自由的行动和决策
财富不是目的，自由才是，财富只是实现自由的手段。追求所爱的自由。
真正的自由不是肆无忌惮，而是掌控力，在一定范围内的选择和拒绝。低维度的自由是拥挤的，难免与其他人发生碰撞；提升掌控力，就是可以跳开争斗，从更高维度求取更多的自由。
4. 为了自我实现的可能
工作通常被认知为赚钱的手段，那么往往是厌烦和劳碌（对抗式），而且是没有希望的（没有发展）
但如果可以剥离其赚钱目的，而只是为了兴趣、热情、成就感和价值感（影响力），当做一个持续成长学习的途径，与人合作交流的契机。是否工作也可以成为一种爱好，一种挑战，一种生活方式，一种人生体验。而这样又是否就真的赚不到钱？

## 学习&工作&生活
学习就是对自己的投资，学习并不是读书-记忆知识点-考试，而是带着疑问、好奇心和探索欲主动得寻求答案，可以是从书上学，可以是从视频学，可以是从生活中的人和事学。学习一定是带着思考思辨的，因为并不一定只有一个标准答案，即使是科学，也存在置信度，需要不断的进行实践和印证。

适合的工作是能给你带来激发活力（积极进取，类似游戏中的求胜心）和思考（主动的思维投入，才能带来意义和价值），唯有如此，才能将个人规划和职业规划进行融合匹配。
在责任感/使命感的驱动和承担挑战的激励下，不断的追求卓越和完美，发现潜在机会和创新解决方案。

即使找不到兴趣、热情、使命感这些，在你迷茫时，工作会推着你前进；在你找不到信念支撑时，工作会让你觉察到你找到社会角色和身份；在你纠缠于生活的琐屑，工作可以让你暂时抽离情绪旋涡；在你学了很多屠龙之术，工作可以让你在实际的产业场景中尝试应用；在你困于一个封闭的小圈子而又无法挣脱之时，工作可以拓展你的社交，并给以归属感和满足感。

生活也就是包括了终身学习，社会实践（也就是工作），个人成长，家庭关系等全部的集合，而在这个集合中，快乐与其并不冲突。我们需要摒弃的是简单的快乐，因为那种瞬时的激素刺激，就像虚拟世界中的狂想幻梦，不会给现实带来任何帮助。

财富自由并非是一种结果，而是一种我们对生活有把控的状态

# 财富是认知的兑现
认知也就是世界观，是你对客观发生事物的观点，态度和判断，进而决定你的行为逻辑，而行为慢慢就会演化成为习惯（也就是慢思考演化为快思考），进而反向改变你的性格，重塑认知。
言辞是世界观的暴露，所谓察言观色，就是观察一个人表达的态度，和对待事情的无意识行为。

财富绝非一蹴而就的，当想要一举暴富时，要么就是赌性上头了，动作就会变形，把命运交于运气，而不是对所有可能发生的情况按概率做应对；要么就会丧失冷静的判断力，被机会的晕轮遮住风险的识别，片面的做出不留后手的决策。

聚焦效应：人在决策或评估时，过度关注某个特定因素而忽视其他重要因素。
比如人容易过分关注自身某种缺陷，仿佛其他人都在盯着自己的缺陷审视，进而影响自己内容本身的表达。

勇士并非感知不到恐惧，而是即使恐惧也能保持镇定进行理智决策，而不是让恐惧主宰行动。

# 如何经营财富
## 获取财富
获取财富的方式无非创造和转移，越基础的越被循环共享使用的，就可以收到更多转移收入（也就是价值杠杆），比如税就是组织提供了社会基建，流通金融等工具后的转移收入。
同理，内容创作是创造收入，而内容平台则更多的是转移收入。
而收入的规模取决于影响力，显然无论创造还是转移都是缺一不可，前者是生产，后者是渠道。

### 被动收入
#### 创建企业
企业就是一个赚钱的飞轮，初期使其转动需要巨大的驱动力，而后便可以凭借惯性，源源不断的产生财富。所要应对的则是赛道和意外的风险。
首先，需要市场调研，了解目标客户群体，以及你的产品或服务在市场中的定位（高端客单价，低端走量）
其次，了解市场竞品，开发产品差异点，制定相应的营销策略（推广规模化）
关系维护：对客户，对大众，对监管
组建高效团队，找合适的人，如何协作激励，让飞轮自动转起来

#### 投资理财
投资的本质就是看公司
看其真实价值（资产债务，业绩估值）和长期增长潜力（新增长点 折现）

公司基本面：财务状况、盈利能力、管理团队、市场地位（是否具有长期竞争优势，也就是护城河）
行业趋势
宏观经济环境：政策导向、经济流动性等

财务报表：
+ 资产负债表
+ 现金流量表
+ 利润表

##### 股票相关知识
股票是是基于股东资格而享有的通过分红获取公司盈利权利的股权凭证
需要通过证券公司在证券交易所，在规定的时间进行买卖

+ 除息（XD）：现金分红后的股价折算
+ 除权（XR）：派股分红后的股价折算

##### 投资策略
分散化：行业领域、市场地域、资产类别
分散集中式：分散在多个行业，每个行业进行仔细分析，挑选最具竞争力和护城河的企业
因为行业具有周期波动性、不同地区可能有不同的政策影响、不同的资产类别则会影响风险和收益率

时间horizon：计划持有投资组合的时间长度

选股：
+ 价值投资：寻找低估的价值洼地，关键在于估值模型（财务报表分析、行业趋势预测、公司治理结构评估等）
+ 成长投资：寻找高成长潜力公司的未来业绩成长

择时：
+ 指数定投：管理费用低
  + 宽基：市场的整体表现，风险分散
  + 行业：强周期性波动，容易受政策、趋势影响，适用于对行业有较为深入了解的投资者
  + 主题：按照概念主题、投资风格偏好等进行筛选的股票。

#### 租赁授权
房产、品牌、知识产权

知识产权（专利商标版权）和品牌授权：一次性费用或持续的特许经营。需要确保合同和协议合法且在有效期内。
可以帮助扩展影响力和市场范围

#### 流量经济
利用网络平台的强大传播力，其本质还是传播学，也就是如何吸引人的注意力，以及如何将其进行高效变现

普通人的生活天然不具备传播要素

##### 注意力
时间就是金钱其实并不成立，有价值的不是时间而是人的注意力，集中注意力的时长会产生相应的收益，时长是一个容器，并非越大就一定装的越多，而注意力则决定了这个容器装入东西的速度和密度

注意力的分配会影响到人的认知和判断，就像人在驾驶一辆车，有信号灯、标牌、路线、路况等多重信息，如果注意力分配出了问题，就可能会有交通事故的风险；另外一个例子就是同样一幅画，可能正着看是一个一个笑脸，倒着看则是一副愁容，这也就影响了你的感知和理解

主动的培养专注力，是将注意力当做肌肉一般进行锻炼
管理内部干扰：焦虑，拖延，自我怀疑
保证专注执行

## 财富管理
### 规划
规划必须要能应对不确定性，比如黑天鹅意外，紧急情况，状态骤变，否则很容易就会被破坏
规划包括固定收入，固定支出，大额消费（长时间折旧），投资（也包括了社交和成长）和风险基金

设定目标的smart原则
specific 具体的：明确而清晰的目标是一切行动的导向，因为只有知道要去哪里才可能行稳而坚
measurable 可量化衡量：可以及时度量和跟踪进度，并可以进行调整（最好可视化）
attainable 可实现的：有方案，有实现路径（任务和步骤拆解）
realistic 实际的：切实可落地，而又具有一定挑战
time-bound 有时限的：有明确时间表，并用里程碑跟进
预算分配是划分资金池：必须消费，可选消费，投资和增长。每个资金池不得侵犯其他资金池额度。

### 负债
负债也是财富的一部分，因为负债是你对应质押物的融资套现，所以不要把清除负债当做目标，而应该维护负债抵押物的价值，包括资产和信用

但不要让自己处于财务困境，因为紧迫和焦虑会影响判断，包括风险和可行性，当然行动也会比较短视容易饮鸩止渴。

### 控制支出
控制支出并不是剪除所有非必要消费以及降低生活品质，因为那样就限制住自己的认知边界，人是不可能赚到认知之外的钱的，只有投入到更多种的经济链条中并深入思考，才可能发现更多机会。
所以真正的省钱是在不会付出太多额外成本（谨防隐形成本）的前提下，花更少的钱达到相似的效能。

#### 审慎的投资消费决策
在行动前问自己几个问题：
1. 是否是真实需求 / 收益和风险是否在可接受的范畴内
2. 当前是否必须 / 当前是否是一个好的介入点
3. 是否可以带来长期的价值或满足感 / 时间horizon：我预期要持有多久
这些问题的答案需要跟自身当前的财务规划不冲突

### 定期重估投资组合
定期根据政治政策、经济产业周期，个人财务状况、风险承受变化，对投资组合进行评估调整

### 财富心态
无论是投资还是事业，难免有波动和低谷

面对资产的暴涨，关键是保持投资消费的审慎态度；
而面对资产的缩水，甚至危机，焦虑、懊恼、自暴自弃都不可能拯救自己，唯有用热爱坚持做正确的有价值的事情，无论大小，才有可能重获新生。

### 长期主义
1. 拥有是“芝麻”还是“西瓜”的判断力（远景视野和洞察）
2. 敢于舍弃“芝麻”的决断力（即使不被理解也能力排众议）
3. 蛰伏等待“西瓜”的耐心和定力（持续的努力和精细化管理，确保足以坚持到可以采摘“西瓜”的时候）

### 复利效应 & 干中学
复利效应就是选择那些可以增厚本金的事情投入精力

如果只是计件完成固定的工作内容，进而获取相应的回报，这种只能带来财富的线性增长；想要获取财富的指数增长必须是工作内容本身就足以提供足够的个人成长。因为拿回报再来换成长就意味着必须投入额外的更多精力来进行，这无疑是花了更多的精力才换到了别人更少精力就取得的，但从ROI就已经不合算，更何况人的精力本身就是有限的，花费更多精力不仅仅意味着效益折损，还意味着必须牺牲休息，亦或是情感。

干中学的最大阻碍就是完美主义，想要毕其功于一役
其实只要勇于迈出第一步，小心试错（避免路径依赖，控制试错风险），敏捷迭代（灵活根据当前情况进行审视改进反思，及时响应变化），积累总结（将碎片知识系统化），就比投入巨大的成本才正式开始要更容易获得更高的目标达成度。
并且单纯的学习往往很难理解一些实践中才能明白的细节和原因（除非花大力气先去记住，再在实践中领悟），这也就是为什么会有纸上谈兵的道理。

学习从本质是就是获取知识、提升认知，那么从书本资料中是学，模仿实践也是学，生活洞察也是学，差别无非是学习的是文字信息、他人经验、经历事物。所以学习其实就是一种认识世界和解决问题的思维方式，于是带脑子的干其实就是学，而反过来，能解决实际问题的学才是真学，也就是学会干。
举个例子，之前一直不明白为什么放养的牛马为什么都要带个铃铛。当遭遇天黑或者旷野，无论是担心牛马撒野冲撞，亦或是视野受限找不到的时候，只要有铃铛的声音，就可以更容易判定牛马的位置和速度，从而解决视觉不足的问题。
抽象一下，发现问题--思考解法--学习他人说书本的解法--解决问题了吗，还能改进吗--总结泛化体系化

善于学习的特点：
+ 好奇心和求知欲
+ 主动学习和积极思考
+ 效率和方法
+ 自律和毅力
+ 持续学习和成长心态
+ 适用性和灵活性
+ 合作和分享

### 各类资产
#### 金融资产
+ 股票：股息分红是其价值锚点，既包括当前的分红，也包括远期可能的分红，也就是盈利能力的提升
  + 普通股：只参与分红，在公司解散清算时，对公司剩余财产的索取权排在债权人和优先股股东之后
  + 优先股：盈利分配时享有优先权，可以按预定比例获得固定股息（在普通股股东分红前进行，所以不受公司业绩波动影响）
+ 债券：固收，但要注意其底层资产风险（偿债能力；市场利率上升，已发行的债券价格可能就会下跌，此时若卖出债券就会遭受损失）
  + 国债
  + 地方政府债
  + 企业债：在债券发行方的资产结构中享有较高的优先权，优先于普通股股东。利率会受到债券市场的流动性影响。
+ 基金：一揽子组合降低风险（分散投资策略），分为开放式（随时申赎）和封闭式（募集期申购，到期赎回）
  + 货币基金：现金存管，投资于存款、同业存单等短期高流动性资产，收益率较低且稳定（可能跑不赢通胀率）。注意是随存随取还是提前支取需要手续费还是利息折损
  + 股票型
  + 债券型
+ 期货：法律协议，承诺未来某个时间以特定价格买卖某种资产。高杠杆率，可以作为对冲策略的工具
+ 期权：买方在未来某个时间以特定价格买入或卖出某个资产的权利的合约（看涨、看跌）。可以作为对冲策略的工具
+ 外汇：高杠杆率，可以作为对冲策略的工具
+ 保险：对可能的意外事件带来潜在损失的对冲，所以在购买之前要明确自己想要对冲的风险是什么（健康医疗、意外事故、财产保全），然后针对性的购买满足需求而又可以进行持续支付的相应产品。
  注意产品的保障范围、费用结构和保费水平（受产品类型、期限、被保人的年龄和健康状况影响）、免赔额、赔付条件和比例、理赔流程；保险公司的规模、信用评级、历史和口碑（响应速度，索赔处理效率）
  保险公司目前一般还提供健康管理服务，如体检、健康咨询、疫苗接种
  关注保险公司的
  + 偿付能力：准备金比例
  + 盈利能力：保险公司也会使用其中一部分保费进行中低风险的投资（房地产、高信用企业债）
  + 再保险安排：保险公司将其承担的风险部分转移给其他保险公司，从而降低自身风险暴露

#### 私募和风险投资
主要投资于非上市公司的股权（一级市场，流动性较差，退出机制较少）
私募（Private Equity）：成长期和成熟期的企业
风投（Venture Capital）：初创期和高成长潜力的企业（市场前景不明朗，商业模式尚未完全验证）
提供资金和资源支持和战略指导，积极参与企业的管理和决策，加速企业发展，增强其市场竞争力
常见的退出机制包括上市（IPO）并购（M&A）再融资等

#### 房地产
流动性较差，租金是其使用价值锚点

#### 黄金
避险资产

#### 基于区块链的虚拟资产
