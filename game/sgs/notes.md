### 周妃
baike_line_ver: 传说

**良姻** 当*每回合首次*有牌移出/入游戏后，你可以与一名其他角色各摸/弃置一张牌，然后你可以令其中一名手牌数为X的角色回复1点体力（X为「箜」数）。
**箜声** 准备阶段，你可以将任意张牌置为「箜」。结束阶段，你获得「箜」中所有非装备牌并令一名角色使用剩余所有「箜」，其失去1点体力。

良姻的加血，必须是你和对方其一，所以要注意：
1. 对自己加血的条件是自己手牌为奇数，然后把一半手牌（上取整）作为「箜」
2. 对对方加血的条件是其手牌数+1作为「箜」
3. 当队友残血时，也可以通过弃牌，把手牌弃至=「箜」中的装备数，则可以让其+1血。比如队友省1张无用手牌或者已经没有手牌，装备也没用，则可以把手牌清空，就能回血。
4. 当你残血时，也需要在弃置牌后让手牌数=「箜」中的装备数，则可以自己+1血。比如「箜」中有1件无用装备，若自己获得「箜」中牌后有2张手牌，则可以其一张手牌，自己回血后，将无用装备给对方替换上，再掉其一血。

良姻跟很多标记类（比如屯田、钟会的权、不屈创、张鲁的米、李通的锋）移出类（比如界陆逊、徐盛、陈琳、程昱、刘封）还有木牛流马都可以有配合

回合内：
1. 组合使用顺序：目标是增加牌量，从而找2张同花色同类型的牌，但【无中】【顺】都是非伤害锦囊，会消耗蹈节，所以先使用基本牌和装备牌，记录已有的伤害锦囊，然后使用这张伤害锦囊，然后记录非伤害锦囊，此时若有奇策，可以转非伤害锦囊，蹈节回收，百出回血
2. 奇策什么锦囊：若目标是回收，就在蹈节未消耗之前，奇策非伤害锦囊
3. 记录什么锦囊：先记录手上的伤害锦囊，再记录后面大概率会奇策的非伤害锦囊
4. 留什么牌：已记录【无懈】优先留之，然后若有已记录对应花色的【闪】【桃】【酒】可以留（酒优先，桃次之，闪最差，草花>黑桃>方>红桃），有红桃杀也可以优先留

回合外：使用【闪】【桃】【酒】【无懈】；被动使用装备
+ 首次使用就增加一次记录机会，非首次使用，就会获得奇策，但正常到你的回合开始就会失去，除非有辅助，否则没什么用
  + 若是以当前回合角色为准：则获得的奇策可以持续到其下回合开始，也就会贯穿你下个回合，可惜OL不是该情况
  + 若是以1号位为准，那么前置位（在当前角色之前）获得奇策同上用不了，后置位（在当前角色之后）可以，可惜OL也不是该情况
+ 【无懈】已记录，则可以回合外过牌或回血；而且可以通过蹈节回收【无懈】
+ 所以，基本牌的作用就是回合外补记录，所以留基本牌可以看着没有记录的花色留
  + 反过来说，回合内用【杀】，就先用黑，再用红桃，最后用方，因为方【闪】最多，留着回合外记录

### 吴苋
1准备2判定3摸牌4出牌5弃牌6结束

结束阶段转出牌阶段，【移荣】摸牌+【移荣】弃牌，然后自然用牌

增加手上限的方法：
1. 回合开始的【穆荫】，条件是不为全场最大
2. 出牌阶段手牌溢出而弃牌
3. 有【桃】且可吃
过牌：以1点手上限为代价，对自己节命

【移荣】的收益模型（X 手牌上限，Y张手牌）：
1. Y < X：手牌较少，只能尽可能的使用手牌（【桃】是否使用取决于X+1是否合适，比如是否会转换摸牌阶段，是否能转换弃牌阶段，是否X超过6；【无中】最好先不用，因为很难2张都能用出去），然后摸X-Y张，上限-1
   1. 若不执行第二次，则收益就是X-Y，代价是上限-1，最大收益是尽可能用光即Y=0，即最大收益为X，除非这X张每个质量都高，或者当前X点数不好，需要主动-1才执行这种情况
   2. 若再尽可能使用至剩余R，则收益是2X-1-Y-R，代价是上限-2，最大收益是R=0（显然R>=Y，所以前提肯定是Y=0），即最大收益为2X-1，仅用于X>=5且用牌能力很好，需要大过牌时
   3. 通常直接弃1，上限恢复，也就是无成本收益X-Y-1张，最大收益是尽可能用光即Y=0，即最大收益为X-1
2. Y > X：手牌较多
   1. 一直使用，则转换为手牌少的情况。仅用于X点数不好，且每张牌都不想弃；或者X>=5且用牌能力很好
   2. 尽可能使用到Y=X+1，然后弃Y-X 张，使上限+1
      1. 若不执行第二次，仅用于X 点数不好（比如=3，且全场都不高于3）
      2. 尽可能使用至剩余R张，摸X+1-R张上限恢复，也就是无成本收益为X+1-R-(Y-X)=2X+1-R-Y，显然当Y和R越小收益越大，也就是说增加用牌能力是的Y=X+1，R=0，此时最大收益为X 张
【贵相】转化出牌阶段的收益点：
1. 多出一张【杀】
2. 使用【移荣】过牌

0手上限：必须使用【穆荫】加到1
1. 1手上限：最好用【穆荫】加到2，除非有4张牌且包括【杀】【桃】，才有必要多出一张【杀】赚收益
2. 2手上限：
  1. 2-牌：若能保证把牌用到<=1张，则用【穆荫】加到3；否则有【桃】也不要吃，尽量不要占用摸牌阶段（除非被兵且不可解）
  2. 3+牌：若有2+【桃】，则不用【穆荫】，吃到4上限，过牌找【杀】
3. 3手上限：
   1. 3-牌：最好使用【穆荫】加到4，除非双【杀】战略意义很大，类比【神速】
   2. 4+牌：除非有【杀】有【桃】，且用牌比较多才有出牌价值，否则还是应当使用【穆荫】
4. 4手上限：能用【穆荫】最好，这时就不要吃【桃】，否则就尽可能找【桃】【无中】把手牌溢出，从而把上限控到5，跳过弃牌阶段
5. 5、6手上限：此时比较自由了，除非同时中兵+乐，导致没有这两个阶段，就相当于被翻面

### 钟琰
其核心收益点在于【观骨】使用的牌不计入次数，此外就是可以额外使用其他人手里以及牌堆顶的牌

综上：
手上有1字可用牌，优先观1
再看有没有4字牌，次优观4
若自己【杀】溢出，优先观自己，否则则看自己手里的字数牌，考虑观那个对手

1. 最优打法是刷3字牌，观顶3，用3，观对方4，用4（可以用对方的3，用自己的4）

   2. 对方有2字牌，或者有优质牌，（此时可以先尝试顺一下，但尽量不要使其少于2张）则观1用1。顺序是3412
   3. 对方没有好用的牌，但还是有【酒】或【杀】，则观1用1，再观对方4，用其牌。顺序是3414
   4. 以上均不成立，则直接观4。顺序是344
2. 手上有4字牌，观顶4，用4
   1. 若自己收有2字牌，则观其2，用己2
      1. 若对方至少有3张牌且有3字牌，则观顶1用1，顺序是4213
      2. 若牌堆顶3张有3字牌，则观顶3张
         1. 若对方或自己有可用的基本牌，则观1用1，顺序是42314
         2. 否则，直接观最大收尾，顺序是4234
      3. 若对方还有可用的优质牌，则观顶1用1，顺序是421X（X为其手牌上限）
      4. 否则直接观顶4，顺序是424
   2. 若牌堆顶还有可用的牌，则观其1用己1，顺序是414
   3. 否则，直接观其上限，尽可能用之，顺序是44
3. 手上有2字牌，观顶2，用2；再观对方4
   1. 若其有4，用4
      1. 若其还有可用的牌，则观顶1用1，再观其上限用之，顺序是241X（X为其手牌上限）
      2. 否则观顶4，顺序是244
   2. 则用其一张关键牌，顺序是24
4. 只能观顶1用1，再观对方4
   1. 若其有4，用4
      1. 若其还有可用的牌，可以赌堆顶2张有2（概率20%） or 3张里有3（概率不足15%），顺序是142X or 143X
      2. 稳妥的一般直接观顶4，顺序是144
   2. 则只能用其一张关键牌，顺序是14

2. 若自己受到属性伤害，重置是发生在体力扣减前，所以是可以触发技能的
**保族** 需要非连环状态才能使用

一般而言，通过两个过牌技，刷连弩和铁索（解锁自己，连上敌人：普通杀可以触发一次【挟术】；属性杀可以触发两次【挟术】）

### 钟繇

### 杨彪
**间难** 出牌阶段开始时，你可以摸两张牌。若如此做，此阶段一名角色失去所有“间难”牌或最后的手牌后，若没有角色处于濒死状态，你令一名角色执行一项：1.弃置两张牌；2.摸两张牌；3.重铸所有装备牌；4.将一张锦囊牌置于牌堆顶或失去1点体力。每回合每个选项限一次。
**义叱** 结束阶段，你可以拼点，若你赢，没赢的角色依次执行“间难”前X个选项（X为你本回合发动“间难”的次数）。
**去惑** 宗族技，当你每回合不因使用、打出或弃置失去手牌后，若其中首次有【酒】或装备牌，你可以令一名同族角色回复1点体力。

主要看能不能在出牌阶段用掉“间难”牌（不光是摸的2张，还包括选项2/3中的摸牌和重铸牌，都算）
如果可以在出牌阶段让一个人空城也可以触发（可以通过选项1达成）
选项4可以控顶（类比李儒）或者打血

若【间难】触发2次，不要找只有一张牌的敌人拼，也不要找牌很多的敌人拼，除非可以确保不赢

### 杨修
**捷悟** 出牌阶段开始时，你可以令一名角色的手牌此阶段始终对你可见。然后你此阶段使用牌指定其他角色为目标后，你可以展示“捷悟”角色一张手牌，若：两张牌花色相同，你摸一张牌；若此牌本回合以此法展示过，你将你与其之中手牌较多的角色一张手牌置于牌堆顶。
**高视** 结束阶段，你可以依次亮出牌堆顶的牌，直到你亮出了本回合你使用过的牌名，或亮出了至少X张牌（X为本回合你发动“捷悟”次数）。然后你可以使用亮出的牌，若你因此使用了所有亮出牌，你摸两张牌。
**去惑** 宗族技，当你每回合不因使用、打出或弃置失去手牌后，若其中首次有【酒】或装备牌，你可以令一名同族角色回复1点体力。

实测，顺拆对【捷悟】角色使用也是暗的，不是明牌
摸牌的时机是指定目标后（在对方进行响应之前），而后再进行手牌数量的比较决定置顶谁的手牌（若相等则无人置顶）
所以注意：
+ 置顶的牌会被八卦判定掉；
+ 置顶的牌也可能被回合外过牌获得（包括卖血过牌，响应过牌，阵亡过牌等）
+ 对方的响应牌会影响后续的牌数计算；
+ 置顶的牌想控下家的判定，就得提前展示出已经用过的牌，并且的看自己是否要用完所有展示牌（因为用完会过2牌，大概率控不了判定了）
+ 若【捷悟】看自己为了过牌，就尽量不要展示同一张牌两次，因为总归是自己的牌置顶，这样影响过牌，所以最好是展示那些马上就要用掉的牌

放装备或【酒】时，可以回血，所以不要着急用掉，也不要着急吃桃
若展示出来的仅有一张【桃】无法使用，此时若同时有【火攻】，可以火攻自己，相当于弃一张废牌过2牌
若展示出来有【无中生有】，一定要最后使用，因为其也会消耗当前展示的牌，最后用的话，要么消耗的是不可用的牌，要么是堆顶未知牌，都是有收益的
如果对方是回合外过牌，那展示的牌即使被对方过到了，也可以使用（这一点存疑）

本质上的收益点在于放【杀】多一刀，放装备或【酒】能回血，放对方牌类似族钟琰的效果
所以：
计算好可以*指向别人*的牌有几张，设为X（这是关键，决定【捷悟】能发动几次）
1. 过牌流
   适合有连弩使用（此时可以【捷悟】看自己），当然也可以期待过牌刷出来连弩
   尽可能展示与使用牌相同花色，用1摸1；当有多张时，尽可能展示回合外不会被用掉的
   然后进行重复展示，计算好剩余次数，先放用过的牌，再放没用过的牌
2. 控牌流
   适合【捷悟】角色有较多高质手牌，而自己的手牌可以消耗到低于其（X-n的消耗量计算，X为【捷悟】发动次数，也就是指向别人的牌输，n为与之不同花色的牌的首次展示）；为了自己手牌更少，有时候需要放弃相同花色的摸牌，而尽量把自己手牌控少，把对方的手牌控顶
   把其中和【捷悟】角色相同花色的牌放在最后使用
   先置顶清理掉自己手上的废牌（包括卡手的防御牌）
   再置顶对方手里的好牌（如果还有同花色的牌，就可以拿防御牌；如果没有同花色的牌，就拿能用的牌）
3. 赌顶流
   适合手里非【杀】的指定别人的牌较多，因为【杀】是牌堆最多的牌名的牌，而展示的牌不限制使用次数，也不限制使用顺序
   所以有时候可以出牌阶段不出【杀】，追求展示的【杀】来杀

### 陆景

弃置：无效之
*注意：第二下的【杀】既不享受【酒】的加成，也不享受无视防具的特效，就只是一张普通的虚拟杀*

宗族：中乐无装备多摸1？脱光装备多摸1（配合重铸可以摸2） or 让对方多弃1

## 木牛流马
仅当木牛流马从装备区移动到装备区(如【甘露】【巧变】)时，其中的牌跟随木牛流马一起移动。其他情况下这些牌均置入弃牌堆。例如被【反馈】时木牛流马进入手牌区，则其中的牌将置入弃牌堆。（并非弃置，落英不收）
当技能允许玩家将一张手牌当其他牌使用和打出时，也可以选择木牛流马中的牌。例如发动武圣时，可以将一张木牛流马中的牌当杀使用或打出。将多张牌当一张牌使用和打出时(如乱击、丈八蛇矛)，可选择任意张手牌和木牛流马中的牌。(特例：荀攸发动【奇策】时只使用手牌)
